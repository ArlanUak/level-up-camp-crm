# Level Up Camp OS

Готовая CRM для учёта детей и управления IT-лагерем.

В проект уже встроены:

- общая SQLite-база данных;
- авторизация и три роли;
- дети, родители и оплаты;
- потоки;
- посещаемость, оценки, XP и комментарии;
- доходы, расходы и прибыль;
- контент-план и задачи SMM;
- Gemini AI для контент-планов и сценариев;
- Excel-выгрузки;
- адаптивная версия для телефона;
- резервная копия всех данных.

**Создавать Supabase, PostgreSQL или таблицы вручную не нужно.** База создаётся автоматически в папке `data` при первом запуске.

---

## Самый простой запуск на Windows

### 1. Установите Node.js

Нужен Node.js версии 22.13 или новее. Лучше установить актуальную LTS-версию 24.

### 2. Вставьте Gemini API-ключ

Откройте файл `.env` обычным Блокнотом и вставьте ключ после знака `=`:

```env
GEMINI_API_KEY=ВАШ_КЛЮЧ
```

Остальные настройки можно не менять.

### 3. Запустите

Дважды нажмите на файл:

```text
START_WINDOWS.bat
```

При первом запуске он сам:

1. установит зависимости;
2. соберёт приложение;
3. создаст базу данных;
4. откроет CRM в браузере.

Адрес на компьютере:

```text
http://localhost:3001
```

Окно запуска нельзя закрывать, пока CRM используется.

## Аккаунты при первом запуске

| Роль | Логин | Пароль |
|---|---|---|
| Администратор | `admin` | `admin123` |
| Преподаватель | `teacher` | `teacher123` |
| SMM | `smm` | `smm123` |

После входа каждый пользователь может сменить свой пароль в разделе **Настройки**.

### Права доступа

**Администратор:** видит детей, потоки, журнал, финансы, маркетинг и настройки.

**Преподаватель:** видит детей, потоки и журнал. Финансы и маркетинг ему не отдаются сервером.

**SMM:** видит контент-план, задачи и Gemini. Данные детей и финансов ей не отдаются сервером.

Ограничения реализованы на сервере, а не только скрытием пунктов меню.

## Как открыть с телефона

Компьютер и телефон должны быть подключены к одной Wi-Fi-сети.

После запуска в чёрном окне появится адрес примерно такого вида:

```text
http://192.168.1.25:3001
```

Откройте этот адрес на телефоне.

При первом запуске Windows может спросить разрешение для брандмауэра. Разрешите доступ для частных сетей.

Для постоянной работы с телефона компьютер, на котором запущена CRM, должен оставаться включённым.

## Где хранятся данные

База находится здесь:

```text
data/level-up-camp.sqlite
```

Она создаётся автоматически. Все пользователи работают с одной базой.

Дополнительно администратор может открыть **Настройки → Скачать резервную копию**.

Excel-файлы скачиваются отдельно из разделов:

- Дети и журнал;
- Финансы;
- Контент-план.

## Gemini

API-ключ хранится только в `.env` на сервере и не показывается преподавателю или SMM в браузере.

По умолчанию используется:

```env
GEMINI_MODEL=gemini-3.5-flash
```

Для анализа актуальных трендов включён Google Search grounding:

```env
GEMINI_SEARCH=true
```

Если тариф ключа не поддерживает поиск, сервер автоматически повторит запрос без него.

После изменения `.env` перезапустите приложение.

## Важно про размещение в интернете

Сам проект не требует отдельной базы данных. Но для круглосуточного доступа нужен компьютер или сервер, который постоянно включён и сохраняет папку `data`.

На бесплатных хостингах с временной файловой системой SQLite-файл может удаляться после перезапуска. Для постоянного интернет-размещения используйте сервер или тариф с persistent disk/volume.
